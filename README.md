# 💰 MoneyMap – Expense Tracker

MoneyMap is a full-stack web application designed to help users track and manage their daily expenses efficiently. It provides an intuitive interface to add, edit, and delete expenses while offering real-time data synchronization with a backend server.

The application includes an interactive dashboard with visual insights such as monthly trends and category-wise expense distribution, making it easier to analyze spending habits.

The frontend is built using HTML, CSS, and JavaScript, while the backend is developed with Node.js and Express.js. The backend is deployed on Render, and the frontend can be hosted using GitHub Pages.

This project demonstrates full-stack development skills, REST API integration, and deployment of web applications in a real-world environment.
